Important FAQ

What Is the Document Object Model (DOM) ?

The DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents a webpage as a tree-like structure, where each element, attribute, and text is a node. Developers use JavaScript to manipulate these nodes dynamically, allowing updates to content, styles, and structure without reloading the page.

How do you select an element from the DOM ?

To select an element from the DOM, use methods like getElementById for IDs, querySelector for the first CSS match, or querySelectorAll for all matches. You can also use getElementsByClassName or getElementsByTagName to fetch elements by class or tag name.

what is event delegation in the context of the DOM, and why is it useful ?

Event delegation uses a single parent element to handle events for its child elements by leveraging event bubbling, where events propagate from a child to its ancestors. Instead of attaching listeners to each child, a parent listens for events and identifies the source using event.target. This improves performance, simplifies code, and supports dynamic child elements efficiently in the DOM.

how do you manipulate an element's attribute and styles using the DOM ?

To manipulate an element's attributes and styles in the DOM, use methods like setAttribute to modify attributes and getAttribute to retrieve them. Directly change properties like id or src. For styles, use the style property to adjust inline CSS or classList to add/remove CSS classes, allowing dynamic visual changes.
